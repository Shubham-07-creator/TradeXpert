# 🚀 TradeXpert

A modern **full-stack stock trading platform** built using the MERN stack, inspired by real-world trading applications like Zerodha Kite.

---

## 📁 Project Structure

```
backend/   → REST API built with Node.js & Express  
frontend/  → Landing page (React)  
dashboard/ → Trading dashboard (React)  
```

---

## ⚙️ Features

* 🔐 User Authentication (Login / Signup)
* 📊 Interactive Dashboard (Holdings, Orders, Funds)
* 🔄 Secure Logout System
* ⚡ Fast and Responsive UI
* 🌐 Environment-based routing (.env support)

---

## 🛠️ Tech Stack

* **Frontend:** React.js, React Router, Axios
* **Backend:** Node.js, Express.js
* **Database:** MongoDB (Mongoose)

---

## ▶️ Run Locally

### 🔹 Backend

```
cd backend
npm install
npm start
```

### 🔹 Frontend (Landing Page)

```
cd frontend
npm install
npm start
```

### 🔹 Dashboard

```
cd dashboard
npm install
npm start
```
